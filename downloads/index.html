Computational Model Builder

Kitware is developing a graphical user interface to Truchas based on their open source Computational Model Builder (CMB) platform. This is a front-end for setting up a Truchas computational model and generating its input file. While not yet feature complete, it is quite usable for a large number of Truchas applications.

The current version of CMB customized for Truchas can be found here. There are packages for Linux, MacOS, and Windows. Some earlier versions can be found in the “OlderVersions” folder.

See here for MacOS installation instructions.
